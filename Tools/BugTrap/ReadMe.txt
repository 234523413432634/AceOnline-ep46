일반적인 버그트랩 시스템을 구현하였다.
크래쉬가 나면 dmp파일과 시스템 정보, 버전정보를 압축 파일로 만들어 지정된 서버에 전송하는 프로그램이다.

주의사항

나라별로 구성관리자에 있는 속성들이 다르게 적용되어 있기 때문에 각 각 나라에 맞는 구성을 선택하고 빌드를 진행하면된다. 

프로그램이 빌드 파일은 각각의 나라별로 따로 생성된다.

실행파일명 : AceTrap.atm

위의 파일을 에이스 온라인의 리소스가 깔려 있는 폴더에 넣으면 크래쉬 발생 시 이 프로그램을 호출 하여 버그를 리포트 할 수 있도록 동작한다.

인자값은 덤프파일이름(확장자 포함 덤프 파일 풀 패스(디렉토리경로 포함))
EX)C:\\AAA\\Ace.dmp
캐릭터이름, 서버 아이피, 로그인아이디, 로그인패스워드, 해당국가 이다.
